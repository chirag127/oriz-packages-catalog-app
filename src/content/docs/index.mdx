## How this catalog works

- **Auto-discovery** — at build time, every public, non-archived `chirag127/*-npm-pkg` repository on GitHub is enumerated. There is no manual package list anywhere in this repo.
- **Per-package detail pages** — for each discovered repo we fetch the README from `raw.githubusercontent.com`, npm metadata from the registry, GitHub repo metadata, and bundle size from bundlephobia, then emit a Starlight page.
- **Grouping** — each package is routed to one of the 5 groups via keyword rules in [`src/lib/groups.ts`](https://github.com/chirag127/oriz-packages-catalog-app/blob/main/src/lib/groups.ts). New packages added to `chirag127/*` will be auto-grouped on the next nightly rebuild.
- **Rebuild cadence** — nightly at 04:00 IST, plus an immediate rebuild whenever any sibling `*-npm-pkg` repo dispatches a `package-published` event after a release.

## Search

Press <kbd>Ctrl</kbd> / <kbd>⌘</kbd> + <kbd>K</kbd> for full-text search powered by [Pagefind](https://pagefind.app). The index covers every package's README plus its metadata fields.
